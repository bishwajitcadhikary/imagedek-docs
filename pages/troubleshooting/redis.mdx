# Redis Enablement

## Hosting Installation

### CloudLinux Hosting (cPanel)

If you are using cPanel shared hosting, you can install Redis by following the steps below:

1. Login to your cPanel account.
2. Search Select PHP Version and click on it.
3. Select the PHP 8.1 version and click on the Select button.
4. Click on the Apply button.
5. Find the redis extension and check the box next to it.

## Local Installation

### Linux

1. Open your terminal
2. Run the following command to install Redis:
```
sudo apt-get install redis-server
```
3. Run the following command to start Redis:
```
sudo service redis-server start
```
4. Run the following command to check if Redis is running:
```
redis-cli ping
```
5. If you see PONG as the output, it means Redis is running.
6. Run the following command to install the PHP Redis extension:
```
sudo apt-get install php-redis
```

### Windows

1. You can find the installation guide for Windows [here](https://redis.io/docs/getting-started/installation/install-redis-on-windows/) or [here](https://www.geeksforgeeks.org/how-to-install-redis-for-php-in-windows/).
2. Locate the php_redis.dll file and copy it to your PHP extension directory.
3. Add the following line to your php.ini file:
```
extension=php_redis.dll
```
4. Restart your web server.

### Mac

1. Open your terminal
2. Run the following command to install Redis:
```
brew install redis
```
3. Run the following command to start Redis:
```
brew services start redis
```
4. Run the following command to check if Redis is running:
```
redis-cli ping
```
5. If you see PONG as the output, it means Redis is running.
6. Run the following command to install the PHP Redis extension:
```
git clone https://www.github.com/phpredis/phpredis.git
cd phpredis
phpize && ./configure && make && sudo make install
```
7. Add the following line to your php.ini file:
```
extension=redis.so
```
8. Restart your php service
```
brew services restart php
```
9. Run the following command to check if the extension is installed:
```
php -m | grep redis
```

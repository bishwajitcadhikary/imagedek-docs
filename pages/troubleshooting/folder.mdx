# File/Folder Permissions

If you are using a Linux server, you will need to make sure that the files and folders are set to the correct permissions. The following is a list of the permissions that you will need to set:

## Directories
```bash
cd cd public_html # where your public folder is
chmod -R 775 storage/logs
chmod -R 775 storage/framework
chmod -R 775 bootstrap/cache
```

# Mollie

### Step 1: Create a Mollie account
Open your browser and go to [https://my.mollie.com/dashboard/signup?lang=en](https://my.mollie.com/dashboard/signup?lang=en). Fill out the form and click `Create Account`.

### Step 2: Get your API keys
After you've created your account, you'll need to get your API keys. Go to [https://my.mollie.com/dashboard](https://my.mollie.com/dashboard) <br/>
Now, go to `Developers > API keys`. Click `Create API key` button.

### Step 3: Configure your API keys
Go to your website's admin panel and navigate to `Finance Management` > `Gateways`.
Find `Mollie` and click the `edit button`,
or go to [https://yourdomain.com/admin/finance/gateways](https://yourdomain.com/admin/finance/gateways). <br/>
Paste your `API Key`.<br/>
Click `Update Changes` button to save your changes.

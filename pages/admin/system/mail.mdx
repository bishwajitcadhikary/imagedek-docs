# SMTP Configuration

Open your browser and go to the following URL: [https://yourdomain.com/admin/system/mail](https://yourdomain.com/admin/system/mail).

Here you can manage the SMTP configuration for your website. This includes setting up the SMTP server, editing existing configurations, and testing the email functionality.

### Features
1. **SMTP Settings**: View and update the SMTP server settings.
2. **Test Email**: Send a test email to verify the SMTP configuration.
3. **Reset Configuration**: Reset the SMTP settings to default values.

### How to Use
1. Navigate to the `SMTP Configuration` page in your admin panel.
2. Use the `SMTP Settings` section to update the server details (e.g., host, port, username, password).
3. Click the `Test Email` button to ensure the configuration is working correctly.
4. If needed, click the `Reset Configuration` button to restore default settings.
5. Follow the on-screen instructions to save your changes.

Note: Always test the configuration after making changes to ensure emails are sent successfully.

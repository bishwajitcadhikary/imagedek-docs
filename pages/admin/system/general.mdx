# General Settings

Open your admin panel and go to `System Settings` > `General Settings` or access it directly at https://yourdomain.com/admin/system/general

## Company Information
Here you can set your company information. This information will be used in the footer of invoices and quotations.

## Site Information
Here you can set your site information including site name, URL, and other basic configurations.

## Google reCAPTCHA

### Step 1: Get reCAPTCHA keys
1. Create a Google account and go to https://www.google.com/recaptcha/admin/create
2. Create a new reCAPTCHA key
3. Select reCAPTCHA v2 and the `I'm not a robot` checkbox option
4. Click on the `Submit` button

Once completed, you will receive your reCAPTCHA site key and secret key.

### Step 2: Set reCAPTCHA keys
1. Open your admin panel and go to `System Settings` > `General Settings`
2. Navigate to the `Google reCAPTCHA` section
3. Enter your reCAPTCHA site key and secret key
4. Select the `Enable` checkbox
5. Click the `Save Changes` button

## Google Analytics

### Step 1: Get Google Analytics Measurement ID
1. Create a Google account and go to https://analytics.google.com/analytics/web/
2. Create a new Google Analytics account
3. Follow the setup wizard to create a property and data stream
4. Obtain your Google Analytics Measurement ID (format: G-XXXXXXXXXX)

### Step 2: Set Google Analytics Measurement ID
1. Open your admin panel and go to `System Settings` > `General Settings`
2. Navigate to the `Google Analytics` section
3. Enter your Google Analytics Measurement ID
4. Select the `Enable` checkbox
5. Click the `Save Changes` button

## Google Maps

### Step 1: Get Google Maps API Key
1. Create a Google account and go to https://developers.google.com/maps/documentation/javascript/get-api-key
2. Create a new Google Maps API Key
3. Copy your Google Maps API Key

### Step 2: Set Google Maps API Key
1. Open your admin panel and go to `System Settings` > `General Settings`
2. Find the `Google Maps API Key` field
3. Paste your Google Maps API Key
4. Click the `Save Changes` button

## Cookie Consent
1. Open your admin panel and go to `System Settings` > `General Settings`
2. Find the `Cookie Consent` section
3. Select the `Enable` checkbox and configure your cookie consent options
4. Click the `Save Changes` button

## IPInfo API Key

### Step 1: Get IPInfo API Key
1. Visit https://apilayer.com/marketplace/exchangerates_data-api to get your IPInfo API Key
2. Register for an account and subscribe to the service
3. Copy your API key

### Step 2: Set IPInfo API Key
1. Open your admin panel and go to `System Settings` > `General Settings`
2. Find the `IPInfo API Key` field
3. Paste your IPInfo API Key
4. Click the `Save Changes` button

## Disqus Comments

### Step 1: Get Disqus Shortname
1. Visit https://disqus.com/admin/create/ to get your Disqus Shortname
2. Follow the steps to create a new site
3. Copy your Disqus Shortname

### Step 2: Set Disqus Shortname
1. Open your admin panel and go to `System Settings` > `General Settings`
2. Find the `Disqus Shortname` field
3. Paste your Disqus Shortname
4. Select the `Enable` checkbox
5. Click the `Save Changes` button

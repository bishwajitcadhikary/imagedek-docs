# General Settings

Open your admin panel and go to **System Settings** > **General Settings** or [https://yourdomain.com/admin/system/general](https://yourdomain.com/admin/system/general)

## Company Information
Here you can set your company information. This information will be used in the footer of the invoice and the quotation.

## Google reCAPTCHA

### Step 1: Get reCAPTCHA keys
Create a Google account and go to [https://www.google.com/recaptcha/admin/create](https://www.google.com/recaptcha/admin/create) and create a new reCAPTCHA key.
Select reCAPTCHA v2 and select **I'm not a robot** checkbox.
Click on **Submit** button.
[![image](https://www.linkpicture.com/q/Screenshot-from-2023-03-05-14-05-40_1.png)](https://www.linkpicture.com/view.php?img=LPic64044ff9d662a1021985722)

Now, you will get your reCAPTCHA keys.
[![image](https://www.linkpicture.com/q/Screenshot-from-2023-03-05-14-05-41_1.png)](https://www.linkpicture.com/view.php?img=LPic640450829e18e2037257678)

### Step 2: Set reCAPTCHA keys
Open your admin panel and go to **System Settings** > **General Settings** or [https://yourdomain.com/admin/system/general](https://yourdomain.com/admin/system/general) . <br/>
Now, in **Google reCAPTCHA** section and set your reCAPTCHA keys and select **Enable** checkbox.
[![image](https://www.linkpicture.com/q/3_827.png)](https://www.linkpicture.com/view.php?img=LPic6416fbe4baa5b1596725234)

## Google Analytics

### Step 1: Get Google Analytics Tracking ID
Create a Google account and go to [https://analytics.google.com/analytics/web/](https://analytics.google.com/analytics/web/) and create a new Google Analytics account. <br/>
Follow the google analytics setup wizard and get your Google Analytics Tracking ID.

### Step 2: Set Google Analytics Tracking ID

Now, in **Google Analytics** section and set your Google Analytics Tracking ID and select **Enable** checkbox.

## Google Maps

### Step 1: Get Google Maps API Key
Create a Google account and go to [https://developers.google.com/maps/documentation/javascript/get-api-key](https://developers.google.com/maps/documentation/javascript/get-api-key) and create a new Google Maps API Key.
Now, you will get your Google Maps API Key. <br/>
Copy your Google Maps API Key. <br/>

### Step 2: Set Google Maps API Key
Open your admin panel and go to **System Settings** > **General Settings** or [https://yourdomain.com/admin/system/general](https://yourdomain.com/admin/system/general) . <br/>
Find **Google Maps API Key** and paste your Google Maps API Key.
[![image](https://www.linkpicture.com/q/5_269.png)](https://www.linkpicture.com/view.php?img=LPic6416fbe4baa5b1596725234)

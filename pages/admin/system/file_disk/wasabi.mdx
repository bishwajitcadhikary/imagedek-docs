## Step: 2 - Configure the Wasabi File System

### Install the Wasabi File System
Open your browser and go to the [https://yourdomain.com/admin/system/file-disks/create](https://yourdomain.com/admin/system/file-disks/create). Select the **Amazon S3/Wasabi** driver and fill the form information and then click on the **Save** button.

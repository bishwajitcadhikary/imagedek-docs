# Language Manager

Open your browser and go to the following URL: [https://yourdomain.com/admin/system/language](https://yourdomain.com/admin/system/language).

Here you can manage the languages for your website. This includes adding new languages, editing existing ones, deleting languages, and managing translations.

### Features
1. **Languages List**: View all the languages currently available on your website.
2. **Add a New Language**: Add new languages with custom details like name and code.
3. **Edit a Language**: Modify the details of an existing language.
4. **Delete a Language**: Remove languages that are no longer required.
5. **Manage Translations**: Update or add translations for your website.

### How to Use
1. Navigate to the `Language Manager` page in your admin panel.
2. Use the `Languages List` to view and select the language you want to manage.
3. Click the `Add`, `Edit`, `Delete`, or `Manage Translations` button based on your desired action.
4. Follow the on-screen instructions to complete the process.

Note: Always test and preview your changes to ensure the language and translations appear as intended.

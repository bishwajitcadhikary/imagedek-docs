# Features Manager

Open your browser and go to the following URL: [https://yourdomain.com/admin/frontend/features](https://yourdomain.com/admin/frontend/features).

Here you can manage the features on your website. This includes creating new features, editing existing ones, and deleting features you no longer need.

### Features
1. **Features List**: View all the features currently available on your website.
2. **Create a New Feature**: Add new features with custom content and settings.
3. **Edit a Feature**: Modify the content or settings of an existing feature.
4. **Delete a Feature**: Remove features that are no longer required.

### How to Use
1. Navigate to the `Features Manager` page in your admin panel.
2. Use the `Features List` to view and select the feature you want to manage.
3. Click the `Create`, `Edit`, or `Delete` button based on your desired action.
4. Follow the on-screen instructions to complete the process.

Note: Always preview your changes before saving to ensure the feature appears as intended.

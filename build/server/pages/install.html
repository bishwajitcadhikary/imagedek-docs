Installation

Requirements

The Laravel framework has a few system requirements. You should ensure that your web server has the following minimum PHP version and extensions:

  • PHP >= 8.1
  • BCMath PHP Extension
  • Ctype PHP Extension
  • cURL PHP Extension
  • DOM PHP Extension
  • Fileinfo PHP Extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PCRE PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • SymLink PHP File Extension

Installation on Shared Hosting

Step 1: Download & Upload

Download the latest version of ImageAI from your CodeCanyon account and unzip it to your computer.

  1. Documentation
  2. Installable Files

(opens in a new tab) Open the installable_file folder and upload the ImageAI.zip file to your server. You can also upload the ImageAI.zip file to your server using an FTP client or File Manager.

(opens in a new tab)

Step 2: File Setup

1. Upload the ImageAI.zip file to your server
  1. Login to your cPanel account.
  2. Click on the File Manager icon.
  3. Click on the public_html folder.
  4. Click on the Upload button.
  5. Click on the Choose File button.
  6. Select the ImageAI.zip file from your computer. (opens in a new tab)
2. Extract the ImageAI.zip file
  1. Click on the ImageAI.zip file.
  2. Click on the Extract button. (opens in a new tab)
  3. Delete the ImageAI.zip file.

Step 3: Database Setup

1. Create a database
  1. Go to your cPanel account.
  2. Click on the MySQL® Database Wizard. (opens in a new tab)
  3. Give a name to your database.
  4. Note: Don't forget to write down your database name, username, and password. You will need them later.
  5. Click on the Next Step. (opens in a new tab)
2. Create a user
  1. Create a username and password for your database.
  2. Click on the Create User. (opens in a new tab)
3. Grant privileges
  1. Now, click on All Privileges button.
  2. Click on the Make Changes button. (opens in a new tab)
4. Done

🥳🎉Congratulations! You have successfully created a database. 🎉🥳 Note: Don't forget to write down your database name, username, and password. You will need them later. (opens in a new tab)

Step 4: Configure .htaccess file

  1. Go to your cPanel account.
  2. Click on the File Manager icon.
  3. Click on the public_html folder.
  4. Click on the .htaccess file.
  5. Click on the Edit button.
  6. Replace the code with the following code:
  7. Click on the Save Changes button.
# If running PHP 7.3 or higher, enable Litespeed caching
<IfModule LiteSpeed>
    CacheLookup on
</IfModule>

# ----------------------------------------------------------------------
# Rewrite Rules
# ----------------------------------------------------------------------
<IfModule mod_rewrite.c>
    Options +FollowSymLinks
    RewriteEngine On

    RewriteCond %{REQUEST_URI} !^/public/

    RewriteCond %{REQUEST_FILENAME} !-d
    RewriteCond %{REQUEST_FILENAME} !-f

    RewriteRule ^(.*)$ /public/$1
    RewriteRule ^(/)?$ public/$1 [L]
</IfModule>

(opens in a new tab) 🥳🎉Congratulations! You have successfully configured the .htaccess file. 🎉🥳

Step 5: Installation Wizard

1. Open your browser and go to your domain name.

For example: http://yourdomain.com/install (opens in a new tab)

2. Server Requirements
  1. Click on the Check Requirements button. (opens in a new tab)
  2. If you see the green tick mark, it means that your server meets all the requirements.
    (opens in a new tab)
3. Folder Permission
  1. Click on the Check Folder Permission button.
  2. If you see the green tick mark, it means that your server meets all the requirements.
  3. Click on the Verify License button. (opens in a new tab)
4. License Verification
  1. Enter your License Key, Enavto Username.
  2. Click on the Verify License button. (opens in a new tab)
5. Database Setup
  1. Fill in the database details that you have created in Step 3.
  2. Click on the Save & Continue button. (opens in a new tab)
6. Admin Setup
  1. Now, fill in the admin details.
  2. Click on the Save & Continue button. (opens in a new tab) 🥳🎉Congratulations! You have successfully installed the ImageAI.
